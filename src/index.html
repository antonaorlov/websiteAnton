Face Recognition [Not Finsished]

The aim of this project was to develop a user-friendly, responsive website that utilizes face recognition software to identify and highlight the faces in any image that is input by the user. The website also features a registration and login system that stores user information in a PostgreSQL database.

In order to achieve this functionality, I employed a variety of technologies, including Node.js for the backend, React and JavaScript for the frontend, and HTML and CSS for styling. Overall, the project required a combination of technical skills in both frontend and backend development, as well as design and user experience considerations.

Responsive Robots

The purpose of this project was to utilize the APIs provided by https://robohash.org/ to create a dynamic and visually appealing React-based grid layout of robots which allows users to search for specific robots, and as they search, the collection of displayed robots is narrowed until the desired robot is displayed. Allowing for an optimal user experience across a range of device sizes and resolutions.

Javascript React for building user interfaces, HTML and CSS for structuring and styling content, and various npm packages to facilitate the efficient management of dependencies and project workflow. Together, these tools enabled me to create a functional and visually engaging application.
